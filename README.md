## 📦 Installation

### Prerequisites
- Node.js (v14+)
- Python (v3.8+)
- MongoDB
- Docker (optional for deployment)

### Steps
```bash
git clone https://github.com/SHREYANK-RAJ/SIH-2025-Project.git
cd SIH-2025-Project

# Backend setup
cd backend
npm install

# Frontend setup
cd ../frontend
open index.html

# Run backend
cd ../backend
node server.js

# Run ML model
cd ../ml
python model.py
```
